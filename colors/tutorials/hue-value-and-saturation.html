Hue, Value, & Saturation

Hue

In simple terms, hue is just another word for color.

Hues have no tint or shade, and are used as the base color to create other colors by adjusting the value and saturation.

Hues can be expressed as a number between 0 and 359, which represent angular positions around the color wheel, where 0 is red, 120 is green, and 240 is blue.

All other values are colors somewhere in between.
